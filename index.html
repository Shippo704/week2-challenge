My Works

Tourist Trap

This is the first collaborative web development project I have done. I worked with two others: Emmanuel Appiagyei and Sivanishanthar Ramachandran. We were tasked with designing a website which used at least two APIs and was user-friendly on various screen sizes. This web application will display weather information and hotel locations in or near whatever location you search for. There is a dropdown menu that will autocomplete various locations based on what the user has typed and updates the list after every character typed. If you can enter the name of a place into google, then this search bar will find it. Here is a link to the repo containing the README file for more information.

Excel Blank Bingo Card Generator

I originally designed this bingo card generator for a friend who does bingo card randomiser runs of the game Hollow Knight. Together, we had brainstormed over 200 prompts to play with various game mods. I have since repurposed the code and remade the game-specific generator into a blank copy for anyone to fill up with their own prompts and generate cards with. The Excel workbook contains instructions for use and has an anti-duplication algorithm built in.
